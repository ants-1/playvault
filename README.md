# PlayVault

PlayVault is a full-stack e-commerce website desiged to deliver a smooth and intuitive online shopping experience. 

## Features

- User Authentication – Secure signup, login, and session handling.

- Product Management – View products, product details, and categories.

- Cart & Checkout – Add to cart, update quantities, and complete orders.

- Image Uploads – Product images managed via Cloudinary.

- Responsive UI – Fully optimized for mobile and desktop devices.

## Tech Stack

**Frontend:**
- React
- Redux
- Chakra UI

**Backend:** 
- Node.js
- Express
- PostgreSQL
- Prisma
- Docker

**Others:** 
- Passport.js (Authentication)
- Zod (Schema Validation)
- Cloudinary + Multer (Image Upload)
- GitHub Actions (CI/CD)
- Jest + Supertest (Testing)

## Roadmap

- **User Dashboard**  
  Includes recent orders and profile update functionality.

- **Admin Dashboard**  
  Full admin panel integration using Admin.js.

- **Frontend Testing**  
  Add unit and integration tests for key UI components and flows.

- **Gamification**  
  Introduce a level-up system based on user purchases to improve engagement.

- **Performance Improvements**  
  Implement Redis caching for most-viewed products to optimize load times.

- **Payment Integration**  
  Add a complete Stripe checkout and payment workflow.
